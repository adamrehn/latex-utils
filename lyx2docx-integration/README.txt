lyx2docx LyX Integration Installer
==================================

This script installs LyX export menu integration for lyx2docx.

To install, place lyx2docx.py in the same directory as the install files, and execute install_lyx2docx.py (or install_lyx2docx.bat under Windows.)

Note that the Windows installer batch script runs install_lyx2docx.py using the version of Python bundled with LyX. In order to locate Python, the script requires that LyX is in the system PATH.
